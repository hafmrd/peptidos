## Consejos para Mejores Resultados

### Midjourney v6
- Usa `--style raw` para menos "artistic interpretation"
- Usa `--no blur, distortion, cartoon, illustration` para evitar estilos no deseados
- Añade `--seed [numero]` para consistencia entre generaciones

### DALL-E 3 (ChatGPT Plus)
- Los prompts funcionan tal cual, son muy obedientes al texto
- Pide "photorealistic product photography" explícitamente
- Especifica "white background, studio lighting"

### Leonardo AI
- Usa modelo "Leonardo Kino XL" o "Leonardo Vision XL"
- PhotoReal activado
- Alchemy activado
- Guidance Scale: 7-9

### Adobe Firefly
- Usa "Photo" como tipo de contenido
- Ajusta "Visual Intensity" a bajo-medio para menos "AI look"
